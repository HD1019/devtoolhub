# The Developer Launch Kit

## 包含内容

### 1. 10个在线工具完整源码 (`/tools-source/`)
纯HTML/CSS/JS实现的完整开发者工具，每个都是独立可部署的单文件：
- JSON格式化 & 验证器
- 图片压缩器 (Canvas API)
- 二维码生成器
- 密码生成器 (crypto API)
- Base64编解码
- UUID生成器
- URL编解码
- 颜色格式转换器
- Markdown实时预览
- 文本差异对比 (Diff)

### 2. 50个AI编程提示词 (`/ai-prompts/`)
为Claude和ChatGPT优化的专业编程提示词，覆盖5大场景。

### 3. 开发者Notion效率模板 (`/notion-templates/`)
5套Notion模板的导入文件。

### 4. SaaS从零到上线实战指南 (`/saas-guide/`)
基于真实项目经验的完整SaaS搭建流程。
