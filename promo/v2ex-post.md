# V2EX 帖子 - 分享创造

**节点**: 分享创造

**标题**: 做了15个完全本地处理的开发者在线工具——不上传、不注册、不追踪

**正文**:

市面上的"免费在线工具"基本都会把你粘贴的JSON/上传的图片发到他们的后端服务器处理。作为一个对隐私有要求的开发者，我受不了这个。

于是花了一天时间写了15个开发者常用工具，核心卖点就一个：**所有处理在浏览器本地完成，数据不离开你的电脑。**

打开F12 → Network标签，使用任何工具，你看不到任何数据上传请求。图片压缩用Canvas API、密码生成用crypto.getRandomValues()、JSON处理用JSON.parse——全是浏览器原生能力。

**工具列表：**
- JSON格式化 & 验证
- 图片压缩（Canvas本地处理）
- 二维码生成（真实可扫）
- 密码生成器（密码学安全随机）
- Base64编解码
- UUID生成器
- URL编解码
- 颜色转换器（HEX/RGB/HSL + 取色器）
- Markdown实时预览
- 文本差异对比（LCS算法逐行diff）
- 字数统计（中英文混合）
- Unix时间戳转换
- HTML实体编解码
- JSON转CSV
- Lorem Ipsum占位文本

**技术栈**: 纯HTML/CSS/JS，每个工具一个独立文件，零框架零构建。唯一的依赖是marked.js（Markdown渲染）。

**部署**: GitHub Pages，0服务器成本。
